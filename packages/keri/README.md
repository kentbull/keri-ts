# keri-ts

KERI TypeScript runtime package with the `tufa` CLI and foundational
database/runtime modules.

## Install

```bash
npm install keri-ts
```

## CLI

```bash
npx tufa --help
npx tufa version
```

## Deno tasks (package-local)

```bash
deno task tufa --help
deno task test:quality
deno task build:npm
```
