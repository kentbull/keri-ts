# keri-ts

KERI TypeScript protocol/runtime library.

## Install

```bash
# As a dependency
npm install keri-ts
```

## Library surfaces

- default import: browser-safe core/types surface
- `keri-ts/runtime`: explicit non-browser-safe runtime surface
- `keri-ts/db`: explicit LMDB-backed storage surface

```bash
npm install keri-ts
```
